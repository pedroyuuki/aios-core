## NOTAS DE PRODUÇÃO

### Prioridade de Teste A/B
1. **C3 (Risco Zero)** — Score Hopkins 9.3/10. Maior diferenciador no mercado.
2. **C2 (Medo Transformado)** — CTR alto em público feminino cristão.
3. **C1 (Chamado + Identidade)** — Headline universal para o nicho.
4. **C5 (Urgência/Transformação)** — Ideal para remarketing.
5. **C4 (Prova Social)** — Mais forte com imagem de depoimentos reais.

### CTA Padrão
Todos usam **"Enviar Mensagem"** → modelo de venda via WhatsApp 1-para-1.
Alternativa: "Saiba Mais" → link `wa.me/{numero}` abre WhatsApp diretamente.

### Regras Técnicas
- Texto overlay: máximo **20% da área** da imagem
- Fontes: **Montserrat Bold** (headlines) / **Lato Regular** (sub) / **Inter Semi-Bold** (CTA)
- Testar legibilidade sempre em mobile
- Resolução mínima: 1080x1350px (4:5) ou 1080x1080px (1:1)

### Fluxo de Produção
1. Gerar imagem com prompt no Midjourney v6 (recomendado) ou DALL-E 3
2. Upscale para resolução final
3. Aplicar texto overlay em Canva / Figma / Photoshop
4. Testar no Meta Ads Preview Tool antes de publicar
